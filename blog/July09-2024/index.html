How to use discord on a tiny ass monitor

July 09, 2024

Discord HATES tiny monitors for whatever reason, so here's how i got mine working

i use discord for an uncomfortable amount of time daily, so much so that i bought a tiny monitor for it so i could play games/stream and also look down and read chat or do whatever. but for some reason discord does not like being under a certain resolution by default and all the UI gets squished. so here's how to mitigate that

Step 1: Adjust discord settings

In user settings, go to Appearance. Try messing with some of these settings, all of these settings already in discord can make a smaller monitor size better to look at (especially zoom level, which is necessary regardless)

Step 1: Edit settings.json

By default, it's in C:\Users\[user]\AppData\Roaming\discord\settings.json. (If you can't find it, it's in the same folder where discord's cache is stored. If you use a different OS, try finding it this way.)

Open it with some text editor. You'll probably see something akin to this!

{ "DANGEROUS_ENABLE_DEVTOOLS_ONLY_ENABLE_IF_YOU_KNOW_WHAT_YOURE_DOING": true, "IS_MAXIMIZED": false, "IS_MINIMIZED": false, "WINDOW_BOUNDS": { "x": -400, "y": 0, "width": 200, "height": 640 }, "MINIMIZE_TO_TRAY": false, "OPEN_ON_STARTUP": false, "debugLogging": false, "MIN_WIDTH": 0, "MIN_HEIGHT": 0, "chromiumSwitches": {}, "enableHardwareAcceleration": true }

It doesn't have to be exactly like mine, but make sure that "MIN_WIDTH" and "MIN_HEIGHT" are both set to 0.

By default, discord limits how small its window can be - setting these values to 0 disables that limitation, so you can put it on any resolution you like!

Also consider adding "DANGEROUS_ENABLE_DEVTOOLS_ONLY_ENABLE_IF_YOU_KNOW_WHAT_YOURE_DOING": true, in there. It sounds intimidating, but it just allows you to use DevTools and other web keyboard shortcuts (like CTRL + SHIFT + I ) inside the discord app

Make sure to save the file and restart discord. Check if you can squish the window as small as you want.

Step 2: Install Vencord or BetterDiscord

Install Vencord here. Install BetterDiscord here.

Both are really easy to install, just download it and go through whatever the installation wizard has to say.

Step 3: Add some custom css

Vencord: put discord on your normal monitor and open User Settings. In the Vencord section chose "Themes", then open the QuickCSS File

BetterDiscord: put discord on your normal monitor and open User Settings. Open the CustomCSS Tab

Once you're in the editor, paste this! @import url("https://suni.me/blog/July09-2024/theme.css");

..that's basically it.

Step 3.5: What is this doing? What if I don't like something?

This does a couple of things:

The server, channel list, and settings lists are compressed into a small bar that slide out on hover. This saves a LOT of valuable screen space.

The toolbar (bar with search) is done in a similar way so that the channel name and part of the description is visible.

Timestamps have been made hidden by default and shown on hover to reduce clutter on the screen.

Usernames are forced to be on one line so that line breaks don't take too much space.

Buttons on message hover have been hidden. I find that it takes up too much space and right clicking the message is easier in the first place.

Vencord's "update now!" notif is now a fixed size because vw had some scaling issues.

The gift nitro and sticker picker buttons on the chatbar are hidden. I find that it takes up too much space on the chatbar.

The rest of the buttons are hidden when you click the chatbar. It also ends up taking too much space.
